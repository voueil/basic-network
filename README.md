# Basic Network Configuration – Cisco Packet Tracer Project

## Overview

This project demonstrates basic computer network using Cisco Packet Tracer

## Technologies & Tools Used
 • Cisco Packet Tracer

## Project Steps
 1. Create a network topology (PCs, Switch, Router)
 2. Connect devices using appropriate cables
 3. Configure IPv4 addresses, subnet masks, and default gateways
 4. Configure router interfaces 
 5. Verify connectivity using the ping command

## Project Contents
 • .pkt file (Packet Tracer project file)

## Requirements
 • Installed Cisco Packet Tracer
 • Basic understanding of IP addressing and network devices
